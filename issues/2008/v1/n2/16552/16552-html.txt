Educadores traduzindo software livre educacional

Autores

  • Rainer Krüger

DOI:

https://doi.org/10.17851/1983-3652.1.2.19-25

Resumo

Este artigo busca mostrar que traduzir um software livre não é tão complicado
quanto parece. É um estímulo aos educadores colaborarem com o desenvolvimento
dos sistemas que usam com seus alunos. Será mostrado quais são os passos para
se traduzir um software, incluindo alguns conceitos como internacionalização,
vocabulário padrão e ferramentas que facilitam o trabalho.
